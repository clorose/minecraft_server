## 🌍 월드 구성

### world (스폰/건축 월드)
- Iris 생성기 사용
- 플레이어 건축, 거주, 경제 활동
- 스폰 중심부: 상점가, 은행, 포탈 허브
- 광석/동굴/구조물 생성 OFF
- 몹 스폰 OFF
- 수면 높이 y=16

### wild (자원 월드)
- 바닐라 생성기 사용
- 자원 채집 전용
- 일반 생성 (모든 것 정상)
- 클레임 불가
- 월 1회 리셋

## 💰 경제 시스템

### 수입원 (돈 버는 방법)
1. **야생 자원** - 광물 채굴 → NPC 판매
2. **낚시** - LiteFish 미니게임 + 통발 (AFK)
3. **농사** - LiteFarm 커스텀 작물
4. **요리** - LiteCooking 버프 음식 제작
5. **특수 도구 제작** - Crucible 드릴/전기톱
6. **퀘스트** - BattlePass 일일/주간 미션
7. **직업 보너스** - Jobs Reborn

### 지출 (돈 쓰는 곳)
1. **수리비** - 도구/방어구 NPC 수리
2. **차량** - 구매 + 연료비
3. **클레임 확장** - 초기 30x30 무료, 이후 유료
4. **텔레포트** - /home, 야생 포탈
5. **인챈트 뽑기** - AdvancedEnchantments
6. **요리 버프** - 소비재
7. **코스메틱** - 칭호, 파티클
