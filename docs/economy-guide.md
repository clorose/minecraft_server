# 🏙️ 메트로폴리스 서버 경제 시스템 가이드

**서버 버전:** 1.21.x
**핵심 플러그인:** Jobs Reborn, ShopGUI+, MythicCrucible, ItemsAdder, LiteFish, Lands, RealisticSeasons

---

## 📖 핵심 철학

> **"혼자서는 모든 걸 가질 수 없다."**

모든 직업은 서로 연결되어 있으며, **1차 생산(재료) → 2차 가공(완제품) → 3차 서비스(유통)**의 순환 구조를 가집니다.

서버 경제는 단순히 "광질해서 팔기"가 아닌, **유저 간 거래와 협력**을 통해 돌아갑니다.

---

## 1. 🌿 1차 산업 : 원자재 생산 (Labor)

> **"땀 흘려 재료를 얻는 자, 부의 기반을 다진다."**
>
> *수익 모델: 서버 상점(ShopGUI+) 판매 60% + 유저 납품 40%*

### 1-1. ⛏️ 광질 (Mining)

| 항목 | 내용 |
|------|------|
| **설명** | 자원 월드에서 광물을 대량으로 채굴. 가장 안정적이고 즉각적인 수입원. |
| **필수 도구** | **[산업용 드릴]** (MythicCrucible) - 3x3 범위 채굴 |
| **수익 포인트** | • **철/구리/돌:** 서버 상점에 즉시 판매<br>• **다이아/에메랄드:** 공학자에게 **강화 재료**로 프리미엄 판매 |

### 1-2. 🌾 농사 (Farming)

| 항목 | 내용 |
|------|------|
| **설명** | 계절(봄/여름/가을/겨울)에 맞는 **특수 작물(커피, 포도, 홉)** 재배. |
| **필수 도구** | **[트랙터]** (ItemsAdder) - 탑승 시 자동 경작 |
| **수익 포인트** | • **[RealisticSeasons]** 때문에 제철이 아니면 작물이 안 자람<br>• **제철 과일**을 창고에 쟁여뒀다가 겨울에 비싸게 파는 **'매점매석'** 전략 가능 |

### 1-3. 🪓 벌목 (Foraging)

| 항목 | 내용 |
|------|------|
| **설명** | 가구 제작에 필수적인 **특수 목재** (흑단, 마호가니, 호두나무) 벌목. |
| **필수 도구** | **[전기톱]** (MythicCrucible) - 나무 전체 파괴(찹트리) |
| **수익 포인트** | • 일반 참나무는 돈이 안 됨<br>• **건축가(Builder)**들이 고급 가구를 만들려면 **[특수 원목]** 구매 필수 (독점 공급) |

### 1-4. 🎣 낚시 (Fishing)

| 항목 | 내용 |
|------|------|
| **설명** | 미니게임을 통해 물고기 포획. 단순 클릭이 아닌 **손맛(컨트롤)** 필요. |
| **필수 도구** | **[루어 낚싯대]** (LiteFish) - 미니게임 난이도 보정 |
| **수익 포인트** | • **도감작(Collection)**을 노리는 수집가들에게 희귀 물고기 판매<br>• 잡은 물고기는 요리사에게 **횟감**으로 납품 |

## 2. 🏭 2차 산업 : 기술 및 가공 (Tech & Craft)

> **"재료에 기술을 더해 가치를 뻥튀기한다."**
>
> *수익 모델: 유저 간 고액 거래 (B2B/B2C) 100%*

### 2-1. 🍳 요리/양조 (Culinary)

| 항목 | 내용 |
|------|------|
| **설명** | 농작물과 해산물을 가공하여 **버프 음식**과 **술(Wine/Beer)** 제조. |
| **필요 재료** | • **농부:** 포도, 홉, 커피<br>• **어부:** 참치, 연어 |
| **수익 포인트** | • **광부용:** 채굴 속도 증가 도시락<br>• **배달부용:** 이동 속도 증가 칵테일<br>• 술은 숙성 기간(현실 시간)이 길수록 가격 폭등 |

### 2-2. 🔧 공학 (Engineering)

| 항목 | 내용 |
|------|------|
| **설명** | 광물을 제련하여 **특수 도구** (드릴, 전기톱, 자동차) 제작. |
| **필요 재료** | • **광부:** 철괴, 구리, 다이아<br>• **벌목꾼:** 고무(수액) |
| **수익 포인트** | • 일반 유저는 '곡괭이'밖에 못 만듦<br>• **드릴**과 **전기톱**은 오직 공학자만 제작/수리 가능 → 높은 마진율 |

### 2-3. 🦴 고고학 (Restoration)

| 항목 | 내용 |
|------|------|
| **설명** | 채굴/낚시 중 얻은 **화석/고대 유물**을 감정하고 복원하여 가치 창출. |
| **필요 재료** | • **광부:** 미감정 화석<br>• **도구:** ItemsAdder 솔 |
| **수익 포인트** | • 쓰레기인 줄 알았던 돌덩이가 감정 후 **[티라노 두개골]**로 판명 → 가격 100배 상승<br>• 부자들의 저택 장식용으로 고가 판매 |

## 3. 🚚 3차 산업 : 서비스 및 유통 (Service)

> **"도시의 시스템을 이용하여 돈을 번다."**
>
> *수익 모델: 수수료, 임대료, 퀘스트 보상*

### 3-1. 📦 배달/운송 (Delivery)

| 항목 | 내용 |
|------|------|
| **설명** | NPC의 의뢰를 받아 물건을 배송. 도시 간 이동하며 무역 차익 실현. |
| **활용 시스템** | • **[탈것]** (오토바이/트럭)<br>• **[BattlePass]** 퀘스트 |
| **수익 포인트** | • 걸어서는 절대 시간 내에 못 감<br>• 좋은 차(스포츠카)를 가진 사람이 배달 랭킹 1위 독점 |

### 3-2. 🏠 부동산 (Real Estate)

| 항목 | 내용 |
|------|------|
| **설명** | 목 좋은 땅(스폰/역세권)을 선점하여 상가 건물을 짓고 세를 놓음. |
| **활용 시스템** | • **[Lands]** 임대 기능<br>• **[ShopGUI+]** 유저 상점 |
| **수익 포인트** | • 접속 안 해도 통장에 임대료가 꽂힘 (불로소득)<br>• 단, 초반에 땅을 살 **막대한 자본** 필요 |

### 3-3. 🎉 핫스팟 (Entertainment)

| 항목 | 내용 |
|------|------|
| **설명** | 고급 미끼를 사용하여 일시적으로 **낚시 명당(Hotspot)** 생성. |
| **활용 시스템** | **[LiteFish]** 미끼 시스템 |
| **수익 포인트** | • "지금 항구에 전설 미끼 뿌립니다! 참가비 500원!"<br>• 유저들을 모아서 파티를 열고 팁을 받음 |

## 4. 💰 보너스 : 숨만 쉬어도 들어오는 돈 (Passive Income)

### 4-1. 직업 급여 (Jobs Reborn)

위의 모든 행동(광질, 제작, 낚시 등)을 할 때마다 시스템이 **추가 월급**을 지급합니다.

```
예시: 다이아 1개 채굴
├── 상점 판매가: 500원
└── Jobs 월급:   +50원
    ─────────────────
    총 수익:      550원
```

### 4-2. 통발 수익 (Fish Trap)

**LiteFish**의 통발을 설치해두면, 학교/직장에 다녀오는 동안 물고기가 자동으로 잡혀 있습니다.

- 소수 인원 서버의 **경제 윤활유** 역할
- 접속 못 해도 최소한의 수익 보장

## 🔄 경제 순환 구조 (The Flow)

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [1차 산업]              [2차 산업]              [3차 산업] │
│                                                             │
│   광부/벌목꾼  ─────────▶  공학자/건축가  ─────────▶  부동산  │
│   (재료 생산)     재료      (도구/가구)      상가      (임대)  │
│       │                        │                      │     │
│       │                        │                      │     │
│       ▼                        ▼                      ▼     │
│   농부/어부   ─────────▶   요리사/양조   ─────────▶  배달부  │
│   (식재료)       납품      (버프 음식)      배송     (운송)   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 순환 예시

1. **광부/벌목꾼**이 재료를 캡니다. → 💵 수익 발생
2. **공학자/건축가**가 그 재료를 사서 드릴/가구를 만듭니다. → 💸 지출 → 💵 더 큰 수익
3. **농부/요리사**는 노동자들에게 버프 음식을 팝니다. → 💵 수익 (효율 증가)
4. **부동산 업자**는 이 모든 사람들이 모일 장터(상가)를 빌려줍니다. → 💵 월세 수익

## 📋 직업별 추천 시작 가이드

| 플레이 스타일 | 추천 직업 | 이유 |
|--------------|----------|------|
| **혼자서 묵묵히** | 광부, 어부 | 서버 상점에 바로 판매 가능, 의존도 낮음 |
| **장사/거래 좋아함** | 요리사, 공학자 | 유저 간 거래로 높은 마진 |
| **접속 시간 적음** | 농부, 통발러 | 패시브 수익 구조 |
| **초반 자본 많음** | 부동산 | 불로소득의 끝판왕 |
| **이동/탐험 좋아함** | 배달부 | 퀘스트 + 운송으로 세계 구경 |

## ⚠️ 주의사항

1. **직업은 동시에 2개까지만** 가질 수 있습니다. (Jobs Reborn 설정)
2. **자원 월드**는 매주 월요일 리셋됩니다.
3. **계절**은 현실 시간 기준 1주일 = 게임 내 1계절입니다.
4. **물가**는 수요/공급에 따라 변동됩니다. (DynamicShop 연동 예정)
